## Проблема
Сайт tezz.uz блокирует загрузку картинок с других доменов (hotlink-защита) — поэтому фото в карточках 15–28 не отображаются, видна только пустая заглушка.

## Решение
1. Скачать фото `https://tezz.uz/uploads/images/product/771/thumbs/208469-10501050.jpg` на сервер.
2. Загрузить его в CDN проекта (Lovable Assets) → получим стабильный URL вида `/__l5e/assets-v1/.../ion-ups.jpg`.
3. Подставить этот URL во все товары с id 15–28:
   - в базе данных (через миграцию `UPDATE public.products`)
   - в локальном fallback-файле `src/data/products.json`

После этого фото будут грузиться у всех посетителей.
